# Pulsar Gaming Shop

**Pulsar Gaming Shop** is a **Mini Ecommerce PRO** project focused on gaming hardware and accessories, developed as part of my professional front-end portfolio.

This project showcases clean front-end architecture, modern UI/UX design, and real-world ecommerce features built with current web technologies.

---

## Live Demo

https://pulsar-gaming-shop.vercel.app/
---

## Features

- Product catalog with categories and search
- Product detail view with image carousel
- Shopping cart functionality:
  - Add and remove products
  - Quantity control
  - Mini product thumbnails
- Checkout experience with:
  - Shipping form validation
  - Multiple payment methods (simulated)
  - Order summary
- Toast notifications
- Responsive design (mobile-first)
- Modern dark UI with smooth animations

---

## Tech Stack

- **React** (Vite)
- **Tailwind CSS**
- **Framer Motion**
- **React Router**
- **JavaScript (ES6+)**
